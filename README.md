# 🌌 Black Hole Simulator 

An interactive 3D web application that visualizes a black hole in space using **Three.js**, combined with a futuristic UI and real scientific concepts like Hawking Radiation.

---

## 🚀 Features

* 🌠 3D starfield with thousands of stars
* 🕳️ Interactive black hole (click to scan)
* 🎯 Raycasting-based object detection
* 📊 Dynamic information panel with physics data
* 🎮 Camera controls (zoom in/out/reset)
* 📡 Transmission form (UI interaction)
* ✨ Sci-fi themed interface

---

## 🧠 Concepts Used

* 3D Graphics with Three.js
* Raycasting (object interaction)
* DOM manipulation (dynamic UI updates)
* Event handling (click, mouse movement, resize)
* Basic astrophysics:

  * Black Holes
  * Event Horizon
  * Hawking Radiation

## 🎮 Controls

* 🔍 **Move Closer** → Zoom into the black hole
* 🔭 **Move Away** → Zoom out
* 🔄 **Reset View** → Reset camera position
* 🖱️ **Click Black Hole** → Scan and display data
* 📡 **Submit Form** → Simulate data transmission

---

## 🧪 How It Works

* A 3D scene is created using **Three.js**
* A sphere represents the black hole
* A torus represents the glowing accretion ring
* Stars are generated using buffer geometry
* Raycasting detects clicks on the black hole
* UI updates dynamically based on interaction
